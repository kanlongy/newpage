## Project Description

This project focused on modernizing the ORB-SLAM3 framework for compatibility with contemporary robotic platforms and demonstrating its versatility across diverse robot morphologies.

**Keywords:** SLAM, ROS2, Robotics, Visual-Inertial Odometry

---

## Key Contributions

- **Framework Modernization:** Updated the ORB-SLAM3 framework for compatibility with Ubuntu 20.04 and ROS2, enabling deployment on contemporary robotic platforms.

- **Multi-Platform Deployment:** Successfully deployed the implementation on diverse robotic platforms including wheeled, aerial, and quadruped robots.

- **Real-World Validation:** Performed real-world localization and mapping tasks, demonstrating robust SLAM performance across different robot morphologies.

- **Integration Enhancement:** Successfully integrated visual-inertial odometry for enhanced tracking accuracy in dynamic environments.
